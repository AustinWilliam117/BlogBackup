**debug 模式下标志位的显示**

| 标志 | 真值为1 | 假值为0 |                        |                   |
| ---- | ------- | ------- | ---------------------- | ----------------- |
| OF   | OV      | NV      | OV = overflow（溢出）  | NV = not overflow |
| SF   | NG      | PL      | NG = negative（负数）  | PL = Positve      |
| ZF   | ZR      | NZ      | ZR=Zero                | NZ = not Zero     |
| PF   | PE      | PO      | PE = EVEN（偶数）      | PO  = ODD         |
| CF   | CV      | NC      | CV = Carry Yes（进位） | NC = not Carry    |
| DF   | DN      | UP      |                        |                   |
